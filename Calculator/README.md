# Modern Neon Calculator

A sleek and modern calculator built using **HTML**, **CSS**, and **JavaScript**.  
This project combines functionality with a visually appealing neon-glow interface, featuring smooth animations, keyboard support, and a fully responsive layout.

---

## Key Features

- **Fully Functional Calculator** – Supports all basic arithmetic operations.  
- **Modern Neon UI** – Elegant glowing effects and smooth transitions.  
- **Keyboard Operable** – Perform all calculations using keyboard keys.  
- **Responsive Design** – Works seamlessly across different screen sizes and devices.  
- **Clean and Modular Code** – Organized structure that is easy to maintain and customize.

---

## Technologies Used

| Technology | Description |
|-------------|--------------|
| **HTML5** | Provides the structural layout of the calculator. |
| **CSS3** | Responsible for the neon styling, layout, and animations. |
| **JavaScript (ES6)** | Handles the calculator logic and user interactions. |

---

## Project Highlights

- Designed with a **neon-glass theme** for a futuristic appearance.  
- Includes **animated button effects** and **real-time display updates**.  
- Features built-in **error handling** (e.g., divide-by-zero protection).  
- Offers both **mouse click and keyboard input** functionality.  
- Developed with a focus on **usability, accessibility, and performance**.
