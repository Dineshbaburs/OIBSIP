
# Ratan Tata – Tribute Webpage

A beautifully designed and responsive tribute webpage dedicated to **Ratan Naval Tata**, one of India’s most respected industrialists and philanthropists.  
This project highlights his life, values, achievements, and inspiring quotes through an elegant, modern, and visually appealing layout.

---

##  Key Features

- **Modern Hero Section** – Full-width banner with overlay text.  
- **Card-Based Layout** – Smooth shadows, hover animations, and clean typography.  
- **Responsive Design** – Works perfectly on mobile, tablet, and desktop.  
- **Image Gallery** – Organized grid showcasing inspiring moments of Ratan Tata.  
- **Smooth UI/UX** – Beautiful spacing, readable fonts, and consistent color theme.  
- **Clean Code Structure** – Well-organized HTML and CSS for easy customization.

---

##  Technologies Used

| Technology | Purpose |
|-----------|---------|
| **HTML5** | Provides semantic structure to the webpage. |
| **CSS3** | Used for layout, styling, animation, and responsiveness. |
| **Google Fonts** | Enhances the visual appearance with modern typography. |
| **Images** | Adds realism and visual storytelling to the page. |

---

##  Project Highlights

- Features a **professional hero banner** with color overlay.  
- Includes structured sections: Introduction, Admiration Points, Values, Achievements, Quotes, and Gallery.  
- Maintains a consistent **blue-toned theme** representing trust and integrity.  
- Image gallery built using responsive CSS Grid.  
- Fully hand-designed layout — no frameworks used.  
- Clean, easy-to-read written content about Ratan Tata.
